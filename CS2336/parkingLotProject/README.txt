This project was written by Abdullah Akbar for his CS2336 (Computer Science II) class at the University of Texas at Dallas

Final Project for the course to demonstrate knowledge of OOP.
Takes in user inputs of various vehicles, then sees if we can park these vehicles in the spots given in our parking lot.

The main file for the program is ParkingLotProblem.cpp.
Input/output example screenshots are in the screenshots folder.
A windows executable of the program is also available in the folder EXE
